## 프로젝트 개요

**DeepRoot**는 플레이어가 WASD로 이동하며 이동 방향으로 화살을 자동 발사하는 원거리 공격 기반 로그라이크 던전 게임입니다.

- 플레이어는 WASD / 방향키로 이동하며 이동 방향으로 자동 화살 발사
- 몬스터는 맵 바깥 4방향에서 스폰되어 플레이어를 향해 이동
- 5마리 처치 → Lv.2, 10마리 처치 → Lv.3 (무기 업그레이드)
- 20마리 처치 → 보스방 자동 전환
- C언어로 게임 로직 구현 → JSON 출력 → 브라우저 UI 연동

## 실행 방법

### UI 실행 (브라우저)

1. VS Code에서 `DeepRoot` 폴더 열기
2. `ui/index.html` 우클릭 → **Open with Live Server**
3. 브라우저에서 **PLAY** 버튼 클릭

### C 백엔드 실행 (Visual Studio)

1. Visual Studio에서 `src/` 폴더의 `.c` `.h` 파일 전부 프로젝트에 추가
2. **Ctrl + F5** 로 실행
3. `data/state.json` 파일에 게임 상태 출력 확인

### 조작 방법

| 키          | 동작                               |
| ----------- | ---------------------------------- |
| `W` / `↑`   | 위로 이동 + 위 방향 발사           |
| `S` / `↓`   | 아래로 이동 + 아래 방향 발사       |
| `A` / `←`   | 왼쪽으로 이동 + 왼쪽 방향 발사     |
| `D` / `→`   | 오른쪽으로 이동 + 오른쪽 방향 발사 |
| 우하단 MUTE | 음소거 토글                        |

## 게임 흐름

```
게임 시작 → PLAY 버튼
    ↓
[MAP 1] 일반 몬스터 전투
    - 10마리 처치 → Lv.2 무기 선택 (단궁 or 석궁)
    - 30마리 처치 → Lv.3 무기 선택
    - 60마리 처치 → 보스방 자동 입장
    ↓
[MAP 2] 보스 전투
    - 보스 HP: 1000
    - 보스 스킬 1: 돌진 (3.5초 주기)
    - 보스 스킬 2: 미사일 (2.5초 주기, 포물선 2발)
    ↓
보스 처치 → 게임 클리어 🎉
HP 0 → 게임 오버 💀
```
